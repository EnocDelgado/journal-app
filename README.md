# Journal App

This is an Journal App, based on Fernando Herrera React's course.

## Installation
Run the following command:
```
yarn
```

## About
This Web Applicationwas created with the following dependencies:
```
1. cloudinary
2. material-icons
3. material-ui
4. reduxjs-toolkit
5. firebase
6. react-redux
7. react-router-dom
8. sweetalert2
```
